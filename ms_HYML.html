CSS 기초 11~20

11. CSS란?

HTML에 스타일을 입혀 색상, 레이아웃, 타이포그래피를 제어합니다.

<link rel="stylesheet" href="styles.css">
12. 선택자(Selector)

element, .class, #id 등으로 HTML 요소를 선택합니다.

/* 태그 선택자 */
p { color: #333; }

/* 클래스 선택자 */
.highlight { background: yellow; }

/* 아이디 선택자 */
#logo { width: 100px; }
13. 색상(Color)과 배경(Background)

색상, 배경 이미지, 배경 크기를 지정할 수 있습니다.

body {
  background: #f0f0f0;
  color: #212529;
}
14. 텍스트 스타일링

폰트 크기, 굵기, 정렬, 줄 높이를 설정합니다.

h1 {
  font-size: 2rem;
  font-weight: bold;
  text-align: center;
}
15. 박스 모델(Box Model)

margin, border, padding, content 구조를 이해합니다.

.box {
  margin: 1rem;
  padding: 1rem;
  border: 1px solid #ccc;
}
16. 레이아웃: Display & Position

block, inline, inline-block, position 속성으로 배치합니다.

.container { display: flex; position: relative; }
.item { position: absolute; top: 10px; left: 10px; }
17. Flexbox

축 정렬, 간격 분배가 쉬운 레이아웃 모듈입니다.

.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
18. Grid

행과 열 단위의 2차원 레이아웃을 만들 수 있습니다.

.grid {
  display: grid;
  grid-template-columns: 1fr 2fr;
  grid-gap: 10px;
}
19. 미디어 쿼리(Responsive)

화면 크기에 따라 다른 스타일을 적용합니다.

@media (max-width: 600px) {
  body { font-size: 14px; }
}
20. 전환(Transition) & 애니메이션(Animation)

부드러운 변화와 반복 애니메이션을 정의합니다.

button {
  transition: background 0.3s;
}
button:hover {
  background: #0056b3;
}

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
.fade-in { animation: fadeIn 1s ease-in; }

JavaScript 기초 21~30

21. JavaScript란?

웹 페이지에 동적인 기능을 추가하는 스크립팅 언어입니다.

<script>
  console.log("Hello, JS!");
</script>
22. 변수와 데이터 타입

let, const 키워드로 변수를 선언하고 숫자, 문자열, 불리언을 사용합니다.

let name = "가나";
const age = 28;
let isStudent = true;
23. 연산자

산술, 비교, 논리 연산자를 사용해 값을 계산하고 판단합니다.

console.log(5 + 3);   // 8
console.log(4 > 2);   // true
console.log(true && false); // false
24. 조건문(if)

if, else if, else로 분기 처리를 합니다.

let score = 85;
if (score >= 90) {
  console.log("A학점");
} else if (score >= 80) {
  console.log("B학점");
} else {
  console.log("C 이하");
}
25. 반복문(for & while)

for와 while문으로 반복 작업을 처리합니다.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

let n = 3;
while (n > 0) {
  console.log(n);
  n--;
}
26. 함수(Function)

재사용 가능한 코드 블록을 정의하고 호출합니다.

function add(a, b) {
  return a + b;
}
console.log(add(2, 3));
27. DOM 조작

document 객체로 요소를 선택하고 콘텐츠를 변경합니다.

let title = document.querySelector("h1");
title.textContent = "안녕하세요!";
28. 이벤트 핸들링

버튼 클릭, 마우스 오버 등의 이벤트를 처리합니다.

let btn = document.querySelector("button");
btn.addEventListener("click", () => {
  alert("버튼 클릭!");
});
29. AJAX & Fetch

서버에서 비동기 데이터를 가져와 화면을 업데이트합니다.

fetch("data.json")
  .then(res => res.json())
  .then(data => console.log(data));
30. 미니 프로젝트: 이미지 슬라이더

간단한 슬라이더를 만들어 setInterval로 자동 전환 기능을 구현합니다.

let imgs = document.querySelectorAll(".slider img");
let idx = 0;
setInterval(() => {
  imgs[idx].classList.toggle("visible");
  idx = (idx + 1) % imgs.length;
  imgs[idx].classList.toggle("visible");
}, 3000);
